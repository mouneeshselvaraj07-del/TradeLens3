TradeLens – AI Powered Seller Assistant,
Project Overview :
TradeLens is a web application that helps online sellers analyze product pricing and estimate profit before listing products on e-commerce platforms.
The application provides visual insights and data analysis to support better pricing decisions.
Problem Statement :
Many online sellers struggle to decide the right selling price for their products.
Wrong pricing decisions can lead to low sales or financial losses.
TradeLens helps sellers make data-driven pricing decisions.
Solution :
TradeLens allows users to enter product details such as cost price and category.
The system analyzes the data and provides insights like:
-Price suggestions
-Profit estimation
-Data visualizations
-Risk analysis
Features :
-User login interface
-Product pricing analysis
-Profit estimation
-Interactive charts and visualizations
-Simple and user-friendly interface
Technology Stack :
-Python
-Streamlit
-Pandas
-NumPy
-Matplotlib
-Seaborn
Installation,
1) Clone the repository :
git clone https://github.com/mouneeshselvaraj07-del/TradeLens3.git
2) Navigate to the project folder :
cd TradeLens3
3) Install dependencies :
pip install -r requirements.txt
4) Run the application :
streamlit run app.py
